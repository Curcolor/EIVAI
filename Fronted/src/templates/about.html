Acerca de EIVAI

¿Qué es EIVAI?

EIVAI es una aplicación de frontend estático desarrollada con FastAPI, diseñada con una arquitectura modular y bien estructurada que facilita el desarrollo y mantenimiento del código.

Características Principales
  • Estructura modular bien organizada
  • Separación clara de responsabilidades
  • Configuración centralizada
  • Rutas organizadas
  • Soporte para archivos estáticos
  • Templates con Jinja2
Tecnologías Utilizadas
  • FastAPI - Framework web moderno
  • Uvicorn - Servidor ASGI
  • Jinja2 - Motor de templates
  • Bootstrap - Framework CSS
  • Font Awesome - Iconos
  • Python - Lenguaje de programación
